## 🇹🇭 ภาษาไทย (Thai)

### 📌 เกี่ยวกับโปรเจกต์นี้
**Windows LTSC by PRP** คือคลังรวบรวมสคริปต์ (Batch/PowerShell), ไฟล์ Registry (.reg) และชุดการตั้งค่าระดับสูงที่ออกแบบมาเพื่อรีดประสิทธิภาพสูงสุดให้กับระบบปฏิบัติการ Windows (โดยเฉพาะเวอร์ชัน LTSC) โปรเจกต์นี้ถูกสร้างขึ้นมาเพื่อเกมเมอร์และผู้ใช้งานแบบ Power User ที่ต้องการลดความหน่วง (Input Lag), เพิ่มเฟรมเรต (FPS), และตัดการทำงานเบื้องหลังที่ไม่จำเป็นออกไป เพื่อให้คอมพิวเตอร์ทำงานได้อย่างรวดเร็วและเสถียรที่สุด

### ✨ ฟีเจอร์หลัก (Features)
* 🛠️ **System & Debloat:** ปิดการทำงานของ Services ที่ไม่จำเป็น, ลบไฟล์ขยะ, ปิด Windows Defender/Update (สำหรับเครื่องที่ต้องการรีดประสิทธิภาพขั้นสุด) และปิด Telemetry
* ⚡ **Custom Power Plans:** แผนการใช้พลังงาน `Ultimate Performance PRP` ที่ปรับแต่งมาเพื่อจ่ายไฟให้ CPU และ GPU อย่างเต็มที่โดยไม่มีสะดุด
* 🎮 **Gaming & FPS Boost:** รวมการตั้งค่าสำหรับเกมโดยเฉพาะ (เช่น `autoexec.cfg` และการตั้งค่าของเกม **Apex Legends**) เพื่อลด Latency และเพิ่มความสมูทในการเล่น
* 🌐 **Network Optimization:** ปรับแต่งค่า TCP, ลด Ping, และปิด Nagle’s Algorithm สำหรับเกมออนไลน์
* 💻 **Hardware Tuning:** สคริปต์ปรับแต่งการทำงานของ CPU Scheduling, การจัดสรร RAM (Memory Management), และ GPU Tweaks (Nvidia Profile Inspector)
* 🎙️ **Streaming Setup:** รวมการตั้งค่า OBS Studio และการตัดเสียงรบกวนของไมโครโฟน

### ⚠️ คำเตือน
การรันสคริปต์ปรับแต่งและแก้ไข Registry อาจส่งผลต่อความเสถียรของระบบ **โปรดทำการสร้าง System Restore Point ก่อนเริ่มใช้งานเสมอ** ผู้จัดทำจะไม่รับผิดชอบต่อความเสียหายใดๆ ที่เกิดขึ้นกับข้อมูลหรือระบบปฏิบัติการของคุณ

## 🇬🇧 English

### 📌 About This Project
**Windows LTSC by PRP** is a comprehensive repository of scripts (Batch/PowerShell), Registry tweaks (.reg), and advanced configurations designed to squeeze the absolute maximum performance out of Windows (specifically tailored for LTSC editions). This project is built for gamers and power users aiming to minimize input lag, boost FPS, and strip away unnecessary background processes for a snappy and stable operating system.

### ✨ Key Features
* 🛠️ **System & Debloat:** Disable unnecessary background services, remove bloatware, disable Windows Defender/Update (for extreme performance tuning), and strip out Telemetry.
* ⚡ **Custom Power Plans:** Includes the `Ultimate Performance PRP` custom power plan, rigorously tuned for uninterrupted CPU and GPU power delivery.
* 🎮 **Gaming & FPS Boost:** Specific game configurations (including `autoexec.cfg` and optimized settings for **Apex Legends**) to lower latency and ensure a buttery-smooth gaming experience.
* 🌐 **Network Optimization:** TCP tuning, Ping reduction scripts, and disabling Nagle’s Algorithm for competitive online gaming.
* 💻 **Hardware Tuning:** Advanced scripts for CPU scheduling, Memory Management allocation, and deep GPU Tweaks (including Nvidia Profile Inspector profiles).
* 🎙️ **Streaming Setup:** Includes OBS Studio settings and microphone noise cancellation configurations.

### ⚠️ Disclaimer
Applying these system tweaks and modifying the Windows Registry can cause system instability or unexpected behavior. **Always create a System Restore Point before proceeding.** Use these scripts entirely at your own risk. The creator is not responsible for any data loss or system damage.
